Goals

The goal of the assignment was to create a website for the Barrier Reef Orchestra. This website was to be an upgraded version of the original website (http://www.nqorchestra.com.au/). The website should be able to attract a new audience of youth for participate in the group as well as some donations to keep the group alive.

Each of the team members were given a webpage to complete. The webpages would then be all put together with a single CSS file and headers/footers/navigation to be used. The team members who did each page are below:
